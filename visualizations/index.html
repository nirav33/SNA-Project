How to Use This Visualization

  • Click on the links above to open different network visualizations
  • In the visualizations:
    • Red nodes represent professors
    • Blue nodes represent coauthors
    • Green nodes represent shared coauthors (connected to multiple professors)
    • Orange edges between professors indicate shared collaborators
    • Hover over nodes for more information
    • Drag nodes to rearrange the network
    • Scroll to zoom in/out
